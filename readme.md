# Harbest.jl
## The best way to scrape the internet
This library combines HTTP, Gumbo and Cascadia for a more simpler way to scrape information. Based on rvest from R.

## Functions

### `read_html`

Read an url

### `html_elements`

Get the elements you want from an html

### `html_text`

### `html_text2`
